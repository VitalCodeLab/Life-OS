# 复盘问题清单（引导有效思考）

## 每日极简复盘 (模式一晚间，模式二可跳过)
*   **核心一问**：（已在晨间记录）今天我为自己设定的核心问题是什么？我是否朝答案前进了一步？
*   **数据瞥见**：看一眼今天的核心指标，有无异常？如有，原因是什么？（30秒思考）

## 每周项目复盘 (周日晚上，20分钟)
1.  **数据回顾**：本周核心指标的平均值、趋势是？与上周比如何？
2.  **假设检验**：我本周的行动，验证或否定了项目立项时的哪个假设？
3.  **障碍识别**：本周最大的执行障碍是什么？（是时间？方法？动力？）
4.  **微调决策**：基于以上，下周需要**坚持**什么？**改进**什么？**停止**什么？
5.  **输出导向**：本周的实践，可以提炼出什么对他人可能有用的“认知碎片”？记入项目笔记。

## 月度系统层复盘 (每月末，60分钟)
1.  **模式分析**：本月有多少天处于“工作主导模式”？主要原因是什么？这对我的人生节奏有何启示？
2.  **项目间关联**：不同项目间有无相互促进或冲突？（如康复训练是否影响了创作精力？）
3.  **框架迭代**：`Life-OS-Framework` 中的哪个模板或协议需要更新？基于什么实践经验？
4.  **方向审视**：我目前所有的项目，是否仍对齐我的长期核心价值？是否需要启动、暂停或归档某个项目？
5.  **下月主题**：为下个月设定一个个人发展的“主题词”（如“恢复”、“连接”、“深化”），并围绕它规划重点。
